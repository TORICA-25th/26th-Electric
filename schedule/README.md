# スケジュール

作業日は基本的に火，水，金とします．  
※永田が居なくても作業することは可としますが，永田が居ない日の作業で起こった事故に対しては責任を負いかねます．

## 凡例
<table>
  <tr>
    <th>記号</th><th>意味</th>
  </tr>
  <tr><td>○</td><td>作業日（作業を与える準備がある日です．来なければならない日ではありません．）</td></tr>
  <tr><td>☆</td><td>招集日（ぜひ来てほしい日です．基本来てください．）</td></tr>
</table>
